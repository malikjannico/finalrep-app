# Sub-orchestrator Task: Milestone 2 - Admin Panel & Associations

## Objective
Implement and verify all requirements under R3 (System Administration) and R4 (Associations & Management) in the FinalRep Streetlifting application.

## Scope
- **R3. System Administration**:
  - Permissions toggle restricting creation of competitions and associations.
  - Application form for users to apply for permissions with reasons.
  - Admin Panel: list users, permission applications, controls to accept/reject/promote users, and configurator for sports, formats, and disciplines.
- **R4. Associations & Management**:
  - Association Creation Wizard (scope, parent association applications, social channels, rulebooks).
  - Association Details view (meets, members, rulebooks, sub-associations).
  - Association Management interface (Owner/Editor roles, ownership transfer, admin approval lists, team lists, competition & athlete groups setup).

## Execution Pattern
You must act as a Sub-orchestrator.
1. Create a `SCOPE.md` in your working directory.
2. Run the iteration loop: Explorer → Worker → Reviewer → gate.
3. Verify using tests.
4. When done, write a `handoff.md` report and send a message back to the Project Orchestrator (conversation ID a99aada5-77f3-425e-8c36-b8635bc01363).
